# Art Brief — Локация «Набережная с книжной лавкой-фургончиком» (v1)

Первая локация проекта. ТЗ художнику для итерации №1. Все ассеты — для мобильной игры в портретной ориентации.

**Дедлайн ориентир:** 5–7 рабочих дней.

---

## 1. Контекст за 30 секунд

Cozy-игра про передвижную книжную лавку. Игрок паркует фургончик-книжарню в разных локациях, продаёт книги покупателям. Стиль — Tiny Bookshop / Spiritfarer / Studio Ghibli backgrounds.

Эта первая локация — **набережная в пасмурный день с лёгким дождём**. Тёплый свет из окна лавки контрастирует с серым морем — главный эмоциональный крюк.

---

## 2. Стилистика — критично

### ✅ Что делаем

**Painterly digital illustration / watercolor.**
- Видимые мазки кисти, мягкие акварельные размывки.
- Лёгкая фактура бумажного зерна по всей картинке.
- Имперфектные линии — как «дрожащая» обводка от руки.
- Палитра пастельная, приглушённая, тёплые vs холодные тона.

### ❌ Что НЕ делаем — это сразу убьёт стиль

- ❌ **Вектор / flat design / Illustrator clean lines** — это другая стилистика (Cult of the Lamb, Stardew portraits). Tiny Bookshop НЕ такой.
- ❌ **Пиксель-арт** — отдельный стилистический мир.
- ❌ **Anime / manga** — даже если в палитре близко.
- ❌ **3D-рендер с toon shader** — выглядит «дёшево».
- ❌ **Фотореализм / hyper detail** — мы делаем «настроение», не «фотографию».
- ❌ **Hard black outlines** — обводки только мягкие, тонкие, цветом темнее основы.

### Стиль-якорь

**Тhree main references:**
1. **Tiny Bookshop** (Neoludic Games) — главный референс. Скриншоты в `docs/INPROGRESS/references/tinybookshop/` (если папки нет — пришлю отдельно).
2. **Studio Ghibli background paintings** — «Унесённые призраками», «Шёпот сердца», «Мой сосед Тоторо» — фоны городов и природы.
3. **Beatrix Potter / детские книжные иллюстрации XX века** — для уюта и наивности линии.

## 3. Композиция кадра — портретная 1080 × 2400

```
y=0       ┌──────────────────────────┐
          │                          │  Sky (~35% высоты)
          │   серое пасмурное небо   │
          │   мягкие облака          │
          │   ░ ░ ░ rain streaks ░ ░ │
y=840     ├──────────────────────────┤
          │▌                         │
          │▌  ░ скала + маячок       │  Средний план (~20%)
          │▌  ~~ горизонт моря ~~    │  Море серо-зелёное,
          │▌  ════════════════════   │  парапет каменный
y=1300    ├──────────────────────────┤
          │                          │
          │     🏮  ┌────────┐       │  Главный объект (~30%)
          │         │ ФУРГОН │ 📦   │  Фургон-лавка по центру
          │    🌻   │ ▓▓▓▓▓▓ │  🪑  │  Декор слева и справа
          │         └────────┘  🌻  │
y=2040    ├──────────────────────────┤
          │    Floor                 │  Мощёная плитка
          │       🐦      🐦         │  Чайки, лужи
y=2280    ╰──── полукруглый бортик ──╯  Foreground frame
y=2400
```

**Принципы:**
- Высокий вертикальный якорь сверху-слева — **флагшток с 3 флагами** (оранжевый, тёмно-синий, бежевый) уходит за верхний край.
- Главный субъект (фургон) — **центр кадра**, занимает ~45% ширины.
- Дальний план светлее, ближний насыщеннее — глубина строится **палитрой**, не размером.
- Foreground frame внизу — полукруглый каменный бортик / низкая клумба — отрезает игрока от сцены.

## 5. Атмосфера и освещение

**Время суток:** пасмурный день, ближе к вечеру.

**Погода:** лёгкий моросящий дождь — диагональные штрихи по всему кадру, **тонкие, белёсые, полупрозрачные**. Не ливень.

**Свет:**
- **Глобальный:** мягкий рассеянный сверху, холодный серо-голубой тон.
- **Тёплый акцент:** **один точечный источник** — окна фургона и фонарик-кашё на стене лавки **горят янтарным**, отбрасывают мягкое пятно на мокрую плитку прямо перед фургоном. Это эмоциональный центр кадра — «тёплый свет в серости».
- **Уличный фонарь** в кадре справа — тоже включён, мягкое свечение вокруг лампы.

**Настроение:** уютная меланхолия. Игрок должен захотеть «зайти внутрь лавки погреться».

## 7. Что НЕ включать в сцену

- ❌ **Покупателей и людей** — только чайки. Покупатели — отдельный набор персонажей, к этой локации не относится.
- ❌ **UI элементы** — никаких кнопок, текста, цифр, иконок интерфейса.
- ❌ **Современные объекты** — автомобили, мотоциклы, провода ЛЭП, фонари с LED, рекламные щиты.
- ❌ **Спидбабблы / диалоговые пузыри**.
- ❌ **Логотипы / водяные знаки художника**.
- ❌ **Жёсткие чёрные тени** — только мягкие, цветные, как акварельные размывки.

## 8. Технические требования

| Параметр | Значение |
|---|---|
| Целевое разрешение игры | **1080 × 2400** (портрет) |
| Рабочий разрешение художника | **2160 × 4800** для full-scene референса, для каждого ассета — 2× от Final size |
| Формат | **PNG, RGBA 8-bit** |
| Фон | **Прозрачный** на всех ассетах с альфа-каналом (фон-плейты — без альфы, плотный JPEG-стиль PNG) |
| Цветовое пространство | **sRGB** |
| DPI | не критично (мы работаем по пиксельным размерам) |
| Слои | **НЕ присылать PSD/Procreate** с группами — только финальные плоские PNG. PSD-исходник держи у себя на случай правок |
| Подпись | **БЕЗ подписи / водяных знаков** в финальном файле |

## 9. Деливерабл — что прислать в финале

1. **Все PNG** по списку §6 в zip-архиве с папочной структурой `promenade/bg/`, `promenade/caravan/`, и т.д.
2. **One composite reference** — `00_full_scene_reference.png` (1080 × 2400) — финальная сцена со всеми элементами на местах. Нужен для проверки общей композиции и атмосферы. **Это композит для эталона**, в игре не используется.
3. **Палитра.png** — извлечённая из финальной сцены палитра как 13 цветных квадратов с hex-кодами. Если палитра уплыла от §4 — это нужно обсудить ДО финальной сдачи.

## 10. Этапы и контрольные точки

| Этап | Что прислать | Когда |
|---|---|---|
| **Чек-поинт 1: Style lock** | Composite reference сцены (или 2–3 варианта на выбор) в финальном стиле, без декора в деталях, но с фургоном и общей атмосферой | День 2 |
| **Чек-поинт 2: Asset pass** | Все 20 PNG в чёрновом качестве, с правильными размерами и композицией | День 4–5 |
| **Финальная сдача** | Все PNG в финальном качестве + composite reference + палитра | День 5–7 |

На каждом чек-пойнте — короткий review (24 часа), могут быть мелкие правки.

## 11. Что важно — резюме одной фразой

> «Уютная меланхолия — фургон-лавка с тёплым светом в окнах припаркован на каменной набережной в моросящий пасмурный день. Хочется зайти и согреться чаем».

Если в финальной картинке этого ощущения нет — что-то пошло не так, обсуждаем до сдачи.
